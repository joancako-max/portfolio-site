# Portfolio Site

Personal portfolio built with React and Vite. Includes links to demo projects and
an HTML CV in `public/cv`.

## CV

- View: `public/cv/Joan-Cako-CV.html`
- Export to PDF: open the HTML in a browser and use **Print** -> **Save as PDF**.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
